# Local AI ChatBot

A production-quality, local ChatGPT-like application built with Streamlit and Ollama.

## Features
- **100% Local**: No cloud APIs, no external inference services.
- **Privacy First**: All chats and models are stored locally on your machine.
- **ChatGPT-like UI**: Dark theme, responsive design, streaming responses.
- **Model Management**: Automatically detects installed Ollama models.
- **Chat History**: Save, load, and delete chat sessions.

## Prerequisites
1. Install Python 3.8+
2. Install [Ollama](https://ollama.com/) and run it locally.
3. Download at least one model using Ollama:
   ```bash
   ollama run llama3
   ```

## Installation
1. Clone or download this project.
2. Install the required Python packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application using Streamlit:
```bash
streamlit run app.py
```

The application will open in your default web browser (usually at `http://localhost:8501`).

## Architecture Prep
The application includes placeholder components for file and image uploads in the sidebar settings, ready for future implementation.
